# Controle de estoque de cestas básicas
Para aprimorar meus conhecimentos em Express.js, MongoDB, sistema MVC e CRUD, criei um sistema focado em auxiliar igrejas e instituições de assistência social. O sistema conta com algumas funcionalidades que facilitam e aprimoram o controle de estoque e de montagem das cestas. O sistema possui as seguintes funções:

- Cadastro de produtos (adicionando a quantidade ou criando novo, caso não exista)
- Visualização de estoque de produtos
- Edição e Remoção de produtos
- Configuração de "Cestas Padrões"
- Alerta de cesta completa quando se tem em estoque a quantidade mínima de cada produto para uma cesta.
- Criação de cestas
- Visualização de estoque de cestas
- Registro de saída de cestas
